## Usage
#### Command-Line Interface
for Help
```bash
$ makeicon --help
```
for create directory with icons from file `Image.png`
```bash
$ makeicons --file Image.png
```
Application Options:
  -f, --file= filename to make assets

Help Options:
  -h, --help  Show this help message
